First installer release for clean Ubuntu and Debian VPS deployments.

- Interactive and unattended one-command installer
- Signed release download and verification
- Mandatory cockpit login with passkey and password + TOTP fallback
- Single-use recovery codes and root-only authentication recovery
- Optional, default-on port knocking with idle session expiry
- Separate, default-off 90-day anonymous telemetry consent
- Complete primary Development environment provisioning
