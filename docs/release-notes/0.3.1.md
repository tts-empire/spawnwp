# SpawnWP 0.3.1

Optional telemetry can now be enabled or disabled from the Updates page. Enabling it
requires explicit confirmation, creates a fresh random installation identifier and
expires automatically after 90 days.

The cockpit explains what is collected and excluded, displays the consent expiry date,
and links to the full privacy notice. The self-hosted receiver stores only the latest
pseudonymous installation state, removes inactive records after 90 days and does not
store request IP addresses.
