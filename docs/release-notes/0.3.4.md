# SpawnWP 0.3.4

This release fixes a bootstrap regression in the runtime stack and broadens installer
compatibility for the next Ubuntu LTS.

- Restores the missing `docker/nginx/default.conf` file used by the WordPress runtime
  container stack.
- Accepts Ubuntu 26.04 in the installer release check so fresh VPS installs can be
  tested on that release.
- Recreates Let’s Encrypt Nginx support files if certbot leaves them absent on a
  reused host.
- Updates the public requirements and marketing copy to match the expanded support
  matrix.
