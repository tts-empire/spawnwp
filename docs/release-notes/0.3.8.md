# SpawnWP 0.3.8

This patch fixes Nginx validation while creating an environment.

- Removes the obsolete `/etc/nginx/cockpit-allowed.conf` include from generated
  Adminer and Mailpit routes.
- Keeps both tools protected by the cockpit's application session through
  `auth_request`.
