# SpawnWP 0.5.12

A small convenience: the cockpit now links to its own documentation.

## Docs link in the header

Every cockpit page (Manage, Deploy, System, Updates) gains a **Docs ↗** link in the header,
next to Log out. It opens the SpawnWP documentation at
[spawnwp.com/docs](https://spawnwp.com/docs/) in a new tab, so the guides — including the new
per-site file browser — are one click away without leaving what you were doing.

## Notes

- Cockpit-only change (HTML + a header style); no backend, dependency or Deploy-plugin
  changes. The link points at the canonical project documentation, which is the same for
  every install.
