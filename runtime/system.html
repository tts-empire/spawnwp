PHP images
Each PHP version keeps one shared image (~1.8 GB) used by all its sites. Keeping an image means every deploy on that PHP version takes about 35 seconds. Deleting one frees its space, but the next deploy on that version rebuilds it from scratch (about 5 minutes). Refresh rebuilds an image now with the latest WordPress — recommended from time to time, since deploys never rebuild automatically. Sites already running keep their current image until their containers are recreated.
